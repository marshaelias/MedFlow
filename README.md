🏥 MedFlow: AI-Powered Medical Workflow Automation
Transforming fragmented medical processes into intelligent, automated workflows using Z.ai GLM orchestration.

🚀 Overview
MedFlow is an agentic workflow engine designed to solve the problem of manual, unstructured medical tasks. Unlike simple chatbots, MedFlow uses Z.ai's GLM as a central "Brain" to interpret patient needs, coordinate specialized worker agents, and maintain stateful medical context across complex procedures.

🧠 The "Brain" (AI Architecture)
A key requirement of this system is that intelligence is the orchestrator, not just a feature.

Central Reasoning Engine: Powered by Z.ai GLM via Ilmuconsole API.

Stateful Execution: The system maintains a "Workflow State Store" to remember context across multi-step interactions.

Agentic Coordination: The Brain dynamically triggers specialized agents:

Triage Agent: Analyzes symptoms and prioritizes cases.

Insurance Validator: Cross-references patient data with simulated policy rules.

Scheduler Agent: Automates appointment booking based on urgency.

✨ Key Features
Visual Reasoning Canvas: A real-time graph showing the AI's step-by-step thinking process.

Thinking Log: A live feed of the agentic reasoning, showing how the GLM handles edge cases like missing data or API failures.

Dynamic Patient Profiling: Extracts unstructured input into structured, actionable medical profiles.

🛠️ Tech Stack
Frontend: React.js for a modern, responsive dashboard.

Backend: Java managing the orchestration logic.

AI Core: Z.ai GLM (integrated via Claude Code and Ilmuconsole).

Tooling: Git/GitHub for version control and collaborative development.
